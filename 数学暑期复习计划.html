01 · 函数综合复习 2次课 · 7/2、7/7

1.1 函数的概念与三要素
  • 函数的定义：非空数集 A→B 的映射，强调 ∀x∈A 有唯一确定的 f(x)∈B
  • 定义域：分式 1/f(x) → 分母 ≠ 0；根式 √f(x) → f(x) ≥ 0；对数 logₐf(x) → f(x) > 0
  • 值域：配方法、单调性法、分离常数法、判别式法、换元法、数形结合法
  • 对应法则：代入法求函数值、待定系数法求解析式、换元法与配凑法
  • 分段函数：求值、解方程/不等式、图像分段、含参讨论
1.2 函数的性质
  • 单调性：定义法（作差/作商）、导数法、图像法；复合函数单调性"同增异减"
  • 奇偶性：定义 f(-x) = f(x) / -f(x)；奇函数 f(0) = 0；常见奇/偶函数举例
  • 对称性：轴对称 f(a-x)=f(a+x) → 关于 x=a 对称；中心对称 f(a-x)=2b-f(a+x)
  • 周期性：定义 f(x+T)=f(x)；常见周期函数模型；周期与对称性的关系
1.3 基本初等函数
  • 指数函数：y=aˣ (a>0, a≠1) 的图像分类（a>1 增/0
  • 对数函数：y=logₐx (a>0, a≠1) 的图像，过定点 (1,0)，与指数函数互为反函数
  • 对数的运算：logₐ(MN)=logₐM+logₐN、logₐ(M/N)=logₐM-logₐN、换底公式 logₐb=logₓb/logₓa
  • 幂函数 y=x^α：不同 α 下的图像特征（α>1、0<α<1、α<0）
  • 指数/对数方程与不等式：同底法、取对数法、图像法
1.4 函数图像与变换
  • 平移变换：y=f(x±a) 左加右减、y=f(x)±b 上加下减
  • 伸缩变换：y=f(ωx) 横向伸缩、y=Af(x) 纵向伸缩
  • 翻折变换：y=|f(x)|（下翻上）、y=f(|x|)（右翻左、去左保右）
  • 对称变换：关于 x轴/y轴/原点/直线 y=x 的对称
  • 函数图像的应用：零点个数、不等式解集、参数取值范围
1.5 函数与方程
  • 零点存在性定理：f(a)·f(b)<0 且连续 ⇒ (a,b) 内有零点（注意：逆命题不成立）
  • 二分法求近似解
  • 零点个数问题：直接法、图像法（f(x)=0 化为两个函数图像交点）
  • 二次函数根的分布：判别式、韦达定理、对称轴位置、端点函数值符号

02 · 三角函数 2次课 · 7/9、7/11

2.1 三角函数的定义与基本关系
  • 任意角的定义：终边相同角 α+2kπ (k∈Z)、象限角与轴线角
  • 弧度制：l=|α|·r（弧长）、S=½lr=½|α|·r²（扇形面积）
  • 三角函数的单位圆定义：sinα=y/r、cosα=x/r、tanα=y/x
  • 各象限三角函数值的符号规律（一全正、二正弦、三正切、四余弦）
  • 同角三角函数关系：sin²α+cos²α=1、tanα=sinα/cosα
2.2 诱导公式与三角恒等变换
  • 诱导公式核心口诀：奇变偶不变，符号看象限（六组诱导公式对应六种角变换）
  • 和差角公式：sin(α±β)=sinαcosβ±cosαsinβ、cos(α±β)=cosαcosβ∓sinαsinβ
  • 倍角公式：sin2α=2sinαcosα、cos2α=cos²α-sin²α=2cos²α-1=1-2sin²α
  • 降幂公式：sin²α=(1-cos2α)/2、cos²α=(1+cos2α)/2
  • 辅助角公式：asinx+bcosx=√(a²+b²)·sin(x+φ)，其中 tanφ=b/a
2.3 三角函数的图像与性质
  • y=Asin(ωx+φ) 的图像：A — 振幅、T=2π/ω — 周期、φ — 初相
  • 五点法作图：选取 ωx+φ = 0, π/2, π, 3π/2, 2π 对应的五个关键点
  • 图像变换顺序：先平移后伸缩 / 先伸缩后平移
  • 求解析式：由最值求 A、由周期求 ω、由关键点求 φ
2.4 解三角形
  • 正弦定理：a/sinA=b/sinB=c/sinC=2R（R 为外接圆半径）
  • 余弦定理：a²=b²+c²-2bc·cosA，变形式 cosA=(b²+c²-a²)/2bc
  • 三角形面积公式：S=½ab·sinC、S=½(a+b+c)r（r 为内切圆半径）
  • 判断三角形形状：正余弦定理结合、勾股定理判定直角三角形
  • 实际应用：仰角俯角、方位角、测量距离/高度问题

03 · 数列 2次课 · 7/14、7/16

3.1 等差数列
  • 定义：aₙ₊₁-aₙ=d（常数）；通项公式 aₙ=a₁+(n-1)d
  • 前 n 项和：Sₙ=n(a₁+aₙ)/2、Sₙ=na₁+n(n-1)d/2
  • 等差中项：2aₙ=aₙ₋₁+aₙ₊₁
  • 性质：若 m+n=p+q 则 aₘ+aₙ=aₚ+a_q；等距项仍等差
  • Sₙ 是关于 n 的二次函数（无常数项），可利用二次函数性质求最值
3.2 等比数列
  • 定义：aₙ₊₁/aₙ=q（q≠0 常数）；通项公式 aₙ=a₁·qⁿ⁻¹
  • 前 n 项和：Sₙ=a₁(1-qⁿ)/(1-q)（q≠1）、Sₙ=na₁（q=1）
  • 等比中项：aₙ²=aₙ₋₁·aₙ₊₁（注意正负号讨论）
  • 性质：若 m+n=p+q 则 aₘ·aₙ=aₚ·a_q
  • 无穷等比数列各项和：S=a₁/(1-q)（|q|<1 时）
3.3 数列求和
  • 公式法：等差数列和、等比数列和、1²+2²+...+n²=n(n+1)(2n+1)/6
  • 裂项相消法：1/[n(n+1)]=1/n-1/(n+1)、1/[(2n-1)(2n+1)]=½[1/(2n-1)-1/(2n+1)]
  • 错位相减法：等差×等比型数列求和（关键步骤——乘公比再相减）
  • 分组求和法：将数列拆分成多个可求和的子数列
  • 并项求和法：相邻两项合并后形成规律
3.4 递推数列
  • aₙ₊₁=aₙ+f(n) 型 → 累加法
  • aₙ₊₁=aₙ·g(n) 型 → 累乘法
  • aₙ₊₁=paₙ+q 型 → 构造法：设 aₙ₊₁+λ=p(aₙ+λ)，化归为等比数列
  • aₙ₊₁=paₙ+q·rⁿ 型 → 两边同除以 rⁿ⁺¹ 构造新数列
  • 递推与数学归纳法结合

04 · 向量 1次课 · 7/18

4.1 向量的基本概念与线性运算
  • 向量的定义：既有大小又有方向的量，记作 \vec{AB} 或 \mathbf{a}
  • 相等向量、共线向量、相反向量、零向量、单位向量
  • 向量的加法：平行四边形法则、三角形法则、多边形法则
  • 向量的减法：三角形法则（差向量指向被减数的终点）
  • 向量的数乘：λ\mathbf{a} 的几何意义（方向与长度变化）
  • 共线定理：\mathbf{a}//\mathbf{b} ⇔ \mathbf{a}=λ\mathbf{b}（b≠0）
4.2 平面向量基本定理与坐标运算
  • 平面向量基本定理：任何向量可由两个不共线的基底唯一线性表示
  • 向量的坐标表示：\mathbf{a}=(x,y)、向量的模 |\mathbf{a}|=√(x²+y²)
  • 坐标运算：加减法对应坐标加减、数乘对应坐标乘以 λ
  • 向量平行（共线）的坐标条件：x₁y₂-x₂y₁=0
4.3 数量积与向量的应用
  • 数量积定义：\mathbf{a}·\mathbf{b}=|\mathbf{a}||\mathbf{b}|cosθ
  • 坐标表示：\mathbf{a}·\mathbf{b}=x₁x₂+y₁y₂
  • 数量积的几何意义：\mathbf{a} 在 \mathbf{b} 方向上的投影 ×|\mathbf{b}|
  • 垂直条件：\mathbf{a}⊥\mathbf{b} ⇔ \mathbf{a}·\mathbf{b}=0 ⇔ x₁x₂+y₁y₂=0
  • 求夹角：cosθ=(\mathbf{a}·\mathbf{b})/(|\mathbf{a}||\mathbf{b}|)
  • 向量的应用：证平行/垂直、求几何长度角度、三角形重心垂心外心向量表示

05 · 解析几何 2次课 · 7/21、7/23

5.1 直线与圆
  • 直线方程的五种形式：点斜式、斜截式、两点式、截距式、一般式 Ax+By+C=0
  • 两直线平行/垂直的条件：k₁=k₂（平行）、k₁·k₂=-1（垂直）
  • 距离公式：两点距离、点到直线距离 d=|Ax₀+By₀+C|/√(A²+B²)、平行线距离
  • 圆的方程：标准式 (x-a)²+(y-b)²=r²、一般式 x²+y²+Dx+Ey+F=0
  • 直线与圆位置关系：d<r 相交、d=r 相切、d>r 相离
  • 弦长公式：|AB|=2√(r²-d²)、圆的切线方程、切点弦方程
5.2 圆锥曲线
  • 椭圆：定义 |PF₁|+|PF₂|=2a；标准方程 x²/a²+y²/b²=1 (a>b>0)；离心率 e=c/a∈(0,1)
  • 双曲线：定义 ||PF₁|-|PF₂||=2a；标准方程 x²/a²-y²/b²=1；渐近线 y=±(b/a)x；e=c/a>1
  • 抛物线：定义 |PF|=d（到焦点距离等于到准线距离）；y²=2px，焦点 (p/2,0)，准线 x=-p/2；e=1
  • 圆锥曲线的统一定义：|PF|/d=e（e=0 圆、0<e<1 椭圆、e=1 抛物线、e>1 双曲线）
5.3 直线与圆锥曲线的位置关系
  • 联立方程法：代入消元 → 一元二次方程 → 判别式判断位置关系
  • 韦达定理的应用：x₁+x₂、x₁x₂ → 弦长公式 |AB|=√(1+k²)·√[(x₁+x₂)²-4x₁x₂]
  • 中点弦问题：设而不求，利用点差法（涉及中点坐标的题型）
  • 焦点弦性质（抛物线）：过焦点的弦，x₁x₂=p²/4、y₁y₂=-p²
  • 定点定值问题：先特殊后一般、设参消参策略
  • 最值与范围问题：建立目标函数 → 判别式法/基本不等式/函数单调性

06 · 立体几何 2次课 · 7/25、7/28

6.1 空间点线面的位置关系
  • 平行判定与性质：线线→线面（a∥b, a⊄α, b⊂α ⇒ a∥α）、线面→面面（a∥β, b∥β ⇒ α∥β）
  • 垂直判定与性质：线线→线面（a⊥b, a⊥c, b∩c=P ⇒ a⊥α）、线面→面面（a⊥α, a⊂β ⇒ α⊥β）
  • 三垂线定理及其逆定理
6.2 空间向量法（核心方法）
  • 建系原则：充分利用已有的垂直关系，选择合适的原点与坐标轴方向
  • 求法向量：利用平面内两个不共线向量叉积的坐标运算
  • 求线线角：cosθ=|a·b|/(|a||b|)（取锐角）
  • 求线面角：sinθ=|a·n|/(|a||n|)（a 为直线方向向量，n 为法向量）
  • 求二面角：cosθ=|n₁·n₂|/(|n₁||n₂|)（需判断锐角还是钝角）
  • 点到面的距离：d=|AP·n|/|n|（A 为面上一点，P 为空间点，n 为法向量）
6.3 空间几何体的度量
  • 柱体：V=Sh（S 底面积，h 高）；S侧=Ch
  • 锥体：V=Sh/3；S侧=½Cl（l 为斜高）
  • 球体：S=4πR²、V=4/3πR³

07 · 导数 2次课 · 7/30、8/1

7.1 导数的概念与运算
  • 导数的定义：f'(x₀)=lim_{Δx→0} [f(x₀+Δx)-f(x₀)]/Δx
  • 导数的几何意义：曲线上某点切线的斜率；切线方程 y-f(x₀)=f'(x₀)(x-x₀)
  • 基本求导公式：(xⁿ)'=nxⁿ⁻¹、(sinx)'=cosx、(cosx)'=-sinx、(eˣ)'=eˣ、(lnx)'=1/x
  • 求导法则：和差积商法则、复合函数求导（链式法则 dy/dx=dy/du·du/dx）
7.2 导数研究函数
  • 单调性：f'(x)>0 ⇒ f(x) 单调增；f'(x)<0 ⇒ f(x) 单调减
  • 极值：f'(x₀)=0 且左右导数异号 → x₀ 为极值点（左正右负为极大，左负右正为极小）
  • 最值：闭区间 [a,b] 上的最值 = max{f(极值点), f(a), f(b)}
  • 导数与函数图像：f'(x) 的符号决定增减、f''(x) 的符号决定凹凸性
7.3 导数的综合应用
  • 恒成立问题：f(x)≥0 恒成立 ⇔ min f(x)≥0（分离参数法：a≥g(x) 恒成立 ⇔ a≥max g(x)）
  • 存在性问题：∃x 使 f(x)≥0 ⇔ max f(x)≥0
  • 函数零点个数：三次函数、含参函数的零点分布（结合单调性与极值符号讨论）
  • 不等式证明：构造辅助函数、利用单调性/最值证明 f(x)≥0
  • 含参分类讨论：确定讨论点（导数为零的点、定义域端点）→ 逐类分析

08 · 概率统计 1次课 · 8/4

8.1 排列组合
  • 分类加法计数原理、分步乘法计数原理（区分"分类"与"分步"）
  • 排列数：A(n,m)=n!/(n-m)!
  • 组合数：C(n,m)=n!/[m!(n-m)!]，性质 C(n,m)=C(n,n-m)、C(n,0)+C(n,1)+...+C(n,n)=2ⁿ
  • 常见计数模型：相邻问题（捆绑法）、不相邻问题（插空法）、定序问题（倍缩法）
  • 分组分配问题：均匀分组 vs 非均匀分组、含名额分配
8.2 二项式定理
  • 展开式：(a+b)ⁿ=C(n,0)aⁿ+C(n,1)aⁿ⁻¹b+...+C(n,n)bⁿ
  • 通项公式：T(r+1)=C(n,r)aⁿ⁻ʳbʳ
  • 二项式系数性质：对称性、最大值（n 为偶数时中间一项最大，奇数时中间两项相等最大）
  • 系数和：令 a=b=1 求系数和、令 a=1,b=-1 求奇偶项系数差
8.3 概率
  • 古典概型：P(A)=m/n（有限等可能）
  • 几何概型：P(A)=d(A)/d(Ω)（长度/面积/体积比）
  • 互斥事件：P(A∪B)=P(A)+P(B)
  • 独立事件：P(AB)=P(A)·P(B)
  • 条件概率：P(B|A)=P(AB)/P(A)
8.4 随机变量与统计
  • 离散型随机变量的分布列、期望 E(X)=Σxᵢpᵢ、方差 D(X)=E(X²)-[E(X)]²
  • 二项分布：X~B(n,p)，E(X)=np，D(X)=np(1-p)
  • 频率分布直方图：组距、频率/组距、平均数与方差的估算
  • 回归分析初步：散点图、相关系数、最小二乘法求回归直线

09 · 不等式 1次课 · 8/6

9.1 不等式的基本性质
  • 传递性：a>b, b>c ⇒ a>c
  • 可加性：a>b ⇔ a±c>b±c
  • 可乘性：a>b, c>0 ⇒ ac>bc；a>b, c<0 ⇒ ac<bc
  • 倒数性质：a>b>0 ⇒ 1/a<1/b
  • 乘方开方：a>b>0 ⇒ aⁿ>bⁿ（n∈N⁺）、a>b>0 ⇒ ⁿ√a>ⁿ√b
9.2 基本不等式
  • 核心公式：(a+b)/2 ≥ √ab（a,b ≥ 0），当且仅当 a=b 时取等
  • 应用条件：一正二定三相等（正数、定值、等号能取到）
  • 常用变形：a+b ≥ 2√ab、a²+b² ≥ 2ab
  • 和定积最大、积定和最小
  • 配凑技巧：添项拆项、整体代换、"1"的代换、分离常数
9.3 一元二次不等式
  • 标准解法：二次项系数化正 → 因式分解/求根公式 → 大于取两边小于取中间
  • 含参讨论：二次项系数符号讨论、判别式讨论、根的大小讨论
  • 恒成立条件：ax²+bx+c>0 恒成立 ⇔ a>0, Δ<0
  • 一元二次不等式的应用：定义域、值域、单调区间等综合问题
9.4 其他重要不等式
  • 分式不等式：移项通分 → 化为一元二次不等式（注意分母不为零）
  • 绝对值不等式：|x|<a ⇔ -a<x<a、|x|>a ⇔ x<-a 或 x>a
  • 绝对值三角不等式：|a|-|b| ≤ |a±b| ≤ |a|+|b|
  • 指数/对数不等式：利用函数单调性（注意底数 a>1 增、0<a<1 减）
  • 柯西不等式初步：(a₁²+a₂²)(b₁²+b₂²) ≥ (a₁b₁+a₂b₂)²

10 · 复数 综合复习课

  • 复数的概念 — 虚数单位 i、i²=-1、复数代数形式 a+bi
  • 复数的四则运算 — 加减乘除（分母实数化）
  • 复数的几何意义 — 复平面、模 |z|=√(a²+b²)
  • 共轭复数 — z̅=a-bi、z·z̅=|z|²
  • 实系数一元二次方程的复数解 — Δ<0 时的共轭虚根
  • 上海高考约 4-5 分，填空选择题，属于必拿的基础分

11 · 综合复习 2次课 · 8/8、8/11

10.1 真题实战
  • 限时完成近 3 年上海高考数学真题（选择填空 40min + 解答题 80min）
  • 批改打分 → 统计各模块得分率 → 定位薄弱环节
  • 典型题精讲：高考常规题型与创新题型的解题策略
10.2 错题回顾
  • 暑期各专题常见错误类型归纳：概念不清类、计算失误类、思路偏差类
  • 函数：定义域遗忘、复合函数拆解错误
  • 三角：诱导公式符号判断、辅助角公式 φ 的确定
  • 数列：错位相减中的项数对齐、裂项后的系数还原
  • 解析几何：联立方程计算不准确、参数范围讨论遗漏
  • 导数：分类讨论的全面性与合理性
10.3 答题策略
  • 考试时间分配：选择填空 40-45min、前三道解答 35-40min、压轴题 25-30min
  • 答题顺序建议：先易后难，遇到卡壳超过 5 分钟先跳过
  • 规范书写：关键步骤不跳步、辅助线/坐标系作图清晰
  • 检查策略：特殊值检验、逆向运算验证、量纲检查
10.4 高三备考衔接
  • 开学后复习节奏预测：9-10月专题强化 → 11-12月一模冲刺 → 1月一模 → 2-3月二轮复习 → 4月二模 → 5月冲刺
  • 整理暑期复习笔记，标注薄弱知识点以备开学后就近查阅
  • 建立错题本（建议电子版，便于检索和分类统计）
  • 推荐教辅：《上海高考数学历年真题》、《高中数学精编》、《上海高考数学一本通》
